# 📺 My Anime Tracker
A custom-built web application to track my anime progress using **Supabase Edge Functions** and the **Kitsu API**.

## 🚀 Features
* **Search & Add:** Search for any anime via the Kitsu API.
* **Live Database:** Powered by PostgreSQL on Supabase.
* **Clean UI:** Simple, responsive grid layout for viewing my collection.

## 🛠️ Tech Stack
* **Backend:** Deno, Supabase Edge Functions
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** PostgreSQL
